# `scripts/` — служебные скрипты

| Скрипт | Назначение |
|--------|-----------|
| `init.sh` | Полный bootstrap: пользователь-песочница, ACL, iptables, запуск всех сервисов (`strace`, `get_sys_info`, Flask), открытие браузера. Параметризуется переменными окружения — см. корневой `README.md`. |
| `stop.sh` | Гасит все фоновые процессы, которые поднял `init.sh`. |
| `kernel_log_reader.py` | `cat /dev/kmsg > /tmp/asda` — нужно для раздела «Сеть» на странице `/logs`. |
| `wait_edits.py` | Отладочная утилита: раз в секунду печатает, изменился ли `/tmp/asd`. |

`init.sh` предполагает, что VSCode-архив лежит в `~/Загрузки/`. Если
путь другой — задайте `VSCEXT_ARCHIVE=/path/to/code.tar.gz`.
